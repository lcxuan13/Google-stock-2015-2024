#  Google-stock-2015-2024
本專案透過 Python 與 Jupyter Notebook，分析 Google 股票從 2015 年至 2024 年的歷史股價與市場波動，並以圖表視覺化呈現。

##  分析內容總覽

**收盤價趨勢圖（Close Price Trend）**
**7、14、30 日移動平均線分析（Moving Average）**
**年度漲跌幅長條圖**
**上漲與下跌年度比例餅圖**
**成交量與波動性對照分析（雙軸圖與 Pair Plot）**
